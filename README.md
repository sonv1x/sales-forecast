# Проєкт: Прогнозування продажів (ML vs Genetic Programming)

Цей репозиторій містить рішення практичного завдання №13. Проєкт порівнює ефективність класичного машинного навчання (XGBoost) та еволюційних методів (Генетичне програмування) для задачі прогнозування попиту в ритейлі.

## Структура репозиторію
- `sales_forecast.ipynb` - Основний Jupyter Notebook з кодом (генерація даних, навчання, графіки).
- `REPORT.md` - Детальний звіт з результатами та бізнес-висновками.
- `xgb_model.pkl` - Навчена модель XGBoost (збережений артефакт).
- `gp_model.pkl` - Навчена модель SymbolicRegressor (збережений артефакт).
- `requirements.txt`- Список бібліотек для запуску.

## Результати
Найкращу точність показала модель **XGBoost** з помилкою MAPE **5.73%**, що значно краще за базовий сезонний прогноз (9.69%).
